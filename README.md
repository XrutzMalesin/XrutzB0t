## Install via Git

```bash
> git clone https://github.com/XrutzMalesin/XrutzB0t
> cd XrutzB0t
> npm start
```

## Install via Zip :

```bash
> apt upgrade && apt update
> pkg install bash
> pkg install git
> cd /sdcard
> cd XrutzB0t-master
> npm start
```
